    36|
    37|- 🛒 **Shopping cart** — add, remove, and update items with persistent state
    38|- 🔍 **Product search** — real-time search with server-side filtering
    39|- 💳 **Stripe checkout** — secure payment processing
    40|- 📱 **Responsive layout** — works on mobile, tablet, and desktop
    41|- 🖼️ **Optimized images** — Next.js Image component with lazy loading
    42|- ⚡ **App Router** — Next.js 15 with React Server Components
    43|- 🧪 **E2E tests** — Cypress test suite running on CI
